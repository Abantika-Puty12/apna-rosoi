# Apna Rosoi - Branding Guidelines

## Brand Identity

### Brand Name
**Apna Rosoi** - "Your Kitchen" in Hinglish (Hindi + English blend)

### Tagline
"Order Delicious, Delivered Fresh"

### Brand Values
- **Quality** - Fresh, quality food from trusted restaurants
- **Convenience** - Easy ordering and fast delivery
- **Trust** - Safe payments and reliable service
- **Community** - Supporting local restaurants

## Color Palette

### Primary Colors
- **Orange (#FF6B35)** - Energy, excitement, appetite
- **Dark Blue (#004E89)** - Trust, reliability, professional

### Secondary Colors
- **Light Gray (#ECF0F1)** - Clean, minimal background
- **Dark Gray (#2C3E50)** - Text, dark elements

### Accent Colors
- **Green (#27AE60)** - Success, vegetarian
- **Red (#E74C3C)** - Alert, error
- **Yellow (#F39C12)** - Warning, rating stars

## Logo Design

### Logo Concept
Fork + Plate + House symbol combined with "Apna Rosoi" text

### Logo Variations

#### Main Logo
```
    🍽️ APNA ROSOI
    [Fork + Plate Icon]
    
Primary Color: Orange (#FF6B35)
Font: Bold, Modern Sans-serif
```

#### Logo Mark (Icon Only)
```
    [Fork + Plate + House icon]
    Versatile for app icon, favicon
```

#### Logo Lockup (Stacked)
```
    [Icon]
    APNA ROSOI
    For vertical layouts
```

### Logo Usage Rules
- Minimum size: 40px width
- Clear space: 20px around logo
- Never distort or rotate
- Always use official colors
- Don't add effects or shadows (unless brand updated)

## Typography

### Fonts
- **Headings**: Segoe UI Bold, Arial Bold
- **Body Text**: Segoe UI Regular, Arial
- **Mono/Code**: Courier New

### Font Sizes
- H1: 2.5rem (40px)
- H2: 2rem (32px)
- H3: 1.5rem (24px)
- H4: 1.25rem (20px)
- Body: 1rem (16px)
- Small: 0.875rem (14px)

## Icon Guidelines

### Icon Style
- **Line weight**: 2px
- **Corners**: Rounded 8px
- **Size**: 24px base (scalable)
- **Color**: Match context (Orange primary, Gray secondary)

### Common Icons
- Home: House
- Order: Bag/Cart
- Track: Pin/Map
- Profile: User
- Settings: Gear
- Back: Left Arrow
- Menu: Hamburger

## Imagery Guidelines

### Food Photography
- High quality, professional photos
- Well-lit, appetizing presentation
- Consistent color palette
- 16:9 aspect ratio preferred

### Restaurant Photos
- Clean, well-lit interiors
- Authentic representation
- At least 1920x1280 resolution
- Professional quality

## Spacing & Layout

### Margin System
- XS: 4px
- SM: 8px
- MD: 16px
- LG: 24px
- XL: 32px
- XXL: 48px

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Tone of Voice

### Writing Style
- **Friendly**: Conversational but professional
- **Clear**: Simple, easy to understand
- **Action-oriented**: Use active verbs
- **Local**: Embrace Indian food culture

### Example Texts
- "Craving something delicious?" (Instead of "Browse restaurants")
- "Your favorite food is just a tap away"
- "Fresh food from local favorites"

## Brand Applications

### Website
- Orange accent buttons
- Clean white backgrounds
- Dark text on light backgrounds
- Consistent spacing

### Mobile App
- Orange header navigation
- Bottom tab bar
- Material Design principles
- Touch-friendly sizes (min 48x48px)

### Email Communications
- Header: Orange bar with logo
- CTA Buttons: Orange primary
- Footer: Dark blue background
- Consistent font family

### Packaging/Physical
- Primary color: Orange
- Secondary: White
- Font: Bold, readable
- Logo: Top right corner

## Brand Violations (DON'T DO)

❌ Change logo colors
❌ Stretch or distort logo
❌ Use unofficial fonts
❌ Mix colors from other palettes
❌ Add shadows or effects to logo
❌ Use low-quality images
❌ Break spacing rules
❌ Use inconsistent button styles
